Требования к библиотеке

Запуск программы - программа должна запускаться из консоли, после запуска пользователю отображается главное меню
Система меню - отображать список доступ. действий, позволяет выбор пункта через ввод числа. Обрабатывать некорректный ввод, повторно запрашивать ввод при ошибке.

Приглашение к вводу - система выводит текст подсказки(объяснять пользователю, что требуется ввести)
Пример: 
Введите логин:
Введите пароль:

Система пользователей:
Регистрация, Вход

Регистрация - выбрать пункт Регистрация, ввод данных логин пароль. Система создает нового пользователя.

Вход - выбрать пункт вход, ввод данных логин пароль. При успешном входе попасть в лич кабинет. При ошибке получить сообщение и повторить ввод.

Личный кабинет - после входа пользователь получает доступ к меню:
просмотр баланса, работа с валютами, работа с долгами, конверсия валют.


Требования к классам:
User - содержит: имя пользователя(string), пароль(string), баланс( набор валют и их кол-во), долги ( набор валют и их кол-во)

Menu - обеспечивает: вывод интерфейса для пользователя, отображение меню, обрабатывает действие пользователя
Функции: интерфейс входа регистрации, интерфейс личного кабинета. Дальше выбор самого функционала: конверсия, работа с долгами, работа с балансом.

Ввод данных: система должна поддерживать разные типы данных: int, double, string
